开发环境搭建

  • VitaSDK — Vita 自制软件的首选解决方案。大多数社区应用、游戏和移植版都是使用此工具链开发的。易于设置和使用，但需要 C/C++ 知识
  • Vita-Development-Suite — 索尼官方 Playstation Vita SDK 的扩展和增强版本。虽然它的使用存在法律上的灰色地带，但在某些开源 SDK 功能不足的情况下，它可能是不可替代的。
  • .
  • -----------图形库---------------------
  • vitagl — 是从 OpenGL 到原生 PS Vita (sceGxm) 图形 API 的转换层。支持大多数需要 OpenGL 的自制游戏和移植版本。
  • PVR_PSP2 — 驱动层 OpenGL 支持 PS Vita。使用尚属法律灰色地带。作为 VitaGL 的替代方案，在某些情况下可能与标准有更好的兼容性。
  • libvita2d — 适用于 PS Vita 的简单 GPU 加速 2D 库（默认随 Vita SDK 安装提供）
  • SDL1 / SDL2 — （默认随 Vita SDK 安装提供）
  • SDL2（Northfear fork） （如果您想从 SDL 管理 VitaGL GL 上下文，请使用）
  • 更多民间开源库

  • -----------语言支持-----------
  • python
  • lpp-vita — PlayStation Vita 制作的 一款lua 解释器，性能较差
  • LIFELUA — PSVita 的 另外一款Lua 解释器,性能更好
  • C#/Mono
